Verification run results

In the Data tab > Verification runs screen, click on View Results in the Actions column to view the performance qualification (PQ) report for a verification run. Managers and Administrators can sign off on this report, at which point it will become a locked PQ Report. However, we recommend that only qualified support specialists sign off on PQ reports.

In addition to basic information about the run, the following data is reported for each of the controls (Ion Dx CB-1 and Ion Dx CF-1) used in the run:

Metric

Description

Base Call Accuracy

1–(total number of errors for all positions in the control / total number of aligned bases).

Key Signal

Average peak signal from incorporation trace measured across the 1-mer incorporations in the control fragment key.

Mean AQ20 read length (bp)

Average length, in base pairs, at which the error rate is ≤1% for all aligned reads of a control fragment.

Percent Reads

Number of all usable library reads that aligned with the control fragment sequence divided by the total number of addressable wells.
