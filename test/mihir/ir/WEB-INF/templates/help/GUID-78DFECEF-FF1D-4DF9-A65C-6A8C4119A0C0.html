Create a copy number baseline

To create a copy number baseline

  1. In the Assay tab > Presets screen > Copy Number Baselines subtab:
  2. Click Create New.

    The baseline creation wizard will open on the Panel page, which has a list of steps across the top. Each step is a new page with different selections and settings.

    • The active step highlighted in gray.

    • Use the Next and Previous buttons on each page to navigate the steps.

  3. In the Panel page:
    1. Enter a name in the CNV Baseline Name field.
    2. Select the panel for which the baseline will be created.
    3. (Optional) Select the Use an existing CNV baseline as a base checkbox, then select a baseline to use as a base for the baseline that you are creating. The new baseline will contain all the samples in the existing baseline, plus any additional samples you add. Note that if you select an existing baseline as a base, the new baseline cannot contain any new samples flagged as "normal."
    4. Add an optional note, then click Next.
  4. In the Set Parameters page, select the appropriate parameters, then click Next.
  5. In the Set Filter page, set the filters, then click Next.
  6. In the Select Samples page, all the samples that have been run with the selected panel will be listed. The minimum number of samples and the number of normal samples required for the baseline will be shown above the list.
    1. Use the checkboxes to select the samples to use for your baseline.
    2. Click the in the Normal Sample column to designate a sample as "Normal" in the baseline.
    3. Click Next.
  7. In the Review Selected Samples page, review your selections, then click Save.
