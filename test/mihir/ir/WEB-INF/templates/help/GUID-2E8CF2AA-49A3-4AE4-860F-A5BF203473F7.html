Managing assays

Click the Assay tab to open the Manage Assays screen.

In the Manage Assays screen, all users (including Operator-level users) can:

  • Search and sort assays

  • View the assay details

  • Create a Reanalysis assay using an existing assay as template

  • View the assay audit trail

Manager and Administrator-level users can also:

  • Create assays (see Create a new assay (Manager/Administrator only))

  • Lock assays

  • Delete assays

  • Obsolete assays

  • Export assays

  • Edit assays

  • Copy assays
