Torrent Suite™ Assay Development Software Help

The results of my run are not showing up in the Data tab

The Ion PGM™ Torrent Server disk space is full

Clear disk space on the Ion PGM™ Torrent Server. Once sufficient disk space has been cleared, data transfer from the Ion PGM™ Dx Sequencer to the Ion PGM™ Torrent Server will commence and data analysis proceed.

Ion PGM™ Dx Sequencer lost connection to the Ion PGM™ Torrent Server

Reestablish the Ion PGM™ Dx Sequencer Ethernet connection to the Ion PGM™ Torrent Server. Once the connection has been restored, data transfer from the Ion PGM™ Dx Sequencer to the Ion PGM™ Torrent Server will recommence and data analysis proceed.
