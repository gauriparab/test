Display message: Failed to set up system time at start up. Check your connection to the Ion PGM™ Torrent Server.

The connection between the Ion PGM™ Dx Sequencer and the Ion PGM™ Torrent Server has been lost

  1. Check the network connection to the Torrent Server to make sure the connection is established, and then reboot the instrument.

  2. If the problem persists, replace the network cable(s) to the instrument and server.

  3. If the problem persists, contact Technical Support (see c_Obtaining support_Ion).

Instrument is still in the process of establishing a connection

Allow ten minutes to see if display message clears.
