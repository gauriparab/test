Data and results

In the Data tab, you can review run results and perform data analysis and data management tasks:

  • Select Completed Runs & Results (the default window) to review completed sample runs and run reports. Results are listed by Sample ID. You can search, filter, and sort the runs to display. You can also reanalyze a sample sequencing run by creating a reanalysis assay with updated analysis parameters, then performing a reanalysis.

  • Select Verification Runs to review data from completed verification runs performed during installation or PQ validation. You can search, filter, and sort the runs to display.
