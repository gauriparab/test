Audit Records (Administrator)

All components in a diagnostic assay must be uniquely identified, and the identification must be stored so that the record can be audited. Within the Audit Records page system Administrators can use the tools provided to perform an audit.

  • Search Audit Records
  • Sort Audit Records
  • Export and print Audit Records
  • Update the Audit Configuration
