Display message: Lost connection to the Ion PGM™ Torrent Server

The connection between the instrument and the server has been lost.

Check the network connection to the Ion PGM™ Torrent Server, and then reboot the Ion PGM™ Dx Sequencer. If this alarm appears during a run, the data created during that run can still be used.
