Prepare a library batch

In Torrent Suite™ Assay Development Software, a library batch consists of a single library or group of libraries, including control libraries, that are prepared at the same time. Control libraries must be included in the same library batch as the sample library they control for.

Note:

Each library must have a unique library name. If you want to combine libraries in the same run, each must also have a unique barcode.

  1. In the Samples tab > Manage Samples screen, click To Be Prepared to display only those samples that have not been placed in a library batch.
  2. Select one or more samples in the list, then click Prepare Library Batch.

    The Prepare Library Batch dialog will open. Required fields are indicated with a red asterisk(*).

  3. In the Select Assay dropdown list, select the appropriate assay. The assay determines certain parameters of the run, including any required controls and post-run data analysis settings.
    Note:

    Only Manager/Administrator level users can create assays.

  4. Enter the barcode from the library kit box if the assay requires the library reagent to be validated.
  5. Type a unique library name for each library in the batch.
    Note:

    If your assay requires controls, they will be automatically listed in the dialog. These controls each require a unique Barcode ID within the library batch but do not require library names.

  6. Select the Barcode ID of the adapter used to prepare each library. If you are combining libraries in the same run, each adapter must have a different Barcode ID.
  7. You may enter the Input Quantity for each library.
  8. Click Save to save your selections and close the dialog.

    The Libraries screen will open, listing the library or libraries that you created. Libraries in the same batch will have the same Library Batch ID.
