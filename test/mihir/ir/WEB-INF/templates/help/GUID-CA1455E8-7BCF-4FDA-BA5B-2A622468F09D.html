Copy number baselines

A copy number baseline (CNV baseline) is a set of control samples that are used to create a baseline for detecting copy number variants (CNVs). They are imported or created in the Assay tab > Presets screen > Copy Number Baseline subtab, and are selected when creating an assay.

Note:

  • A minimum of 48 samples are required to prepare a baseline, including 6 samples of the same gender and flagged as Normal.

  • Assays that use baseline controls can provide better copy number detection (for both CNVs and aneuploidy) than paired sample assays.

  • Copy number estimates are made by counting reads for each amplicon, making adjustments to account for certain types of variability, comparing those read counts to expected counts for those amplicons in a "normal" sample and then making further adjustment.

  • Known sources of variability include pool imbalance (when the assay has more than one pool of amplicons), total number of reads and per amplicon attributes of GC proportion and length of the amplicon insert.
