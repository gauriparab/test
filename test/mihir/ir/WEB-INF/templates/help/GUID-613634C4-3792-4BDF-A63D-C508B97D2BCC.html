Planned Runs

Planned Runs contain all the sample, library, and assay information for performing a run on template preparation and sequencing instruments. Each Planned Run includes a Run Short Code in alphanumeric and barcode formats. By scanning or entering the code, the Planned Run's settings are transferred to the instruments, and each run and its associated information are tracked from template preparation through sequencing, data analysis, and export.
