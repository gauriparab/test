Import an Assay

In the Assay tab > Import Assay screen, Manager- and Administrator-level users can import assays into the software in the form of a .zip file.

  1. In the Import Assay screen, click Select file, select the .zip file to import, and click Open.

    The file name will appear in the Select file field.

    • Click Change to replace the selected file.

    • Click Remove to clear the field.

  2. Click Import to complete the import.

    The assay will appear in the Manage Assays screen.
