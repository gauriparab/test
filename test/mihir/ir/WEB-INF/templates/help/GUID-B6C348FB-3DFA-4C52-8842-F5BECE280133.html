Import samples

In the Samples tab > Import Sample screen, you can import sample data in the form of a .txt, .xls, .xlsx or .csv file. Ensure that the same sample attributes that are entered in the Add new sample dialog are included in the import file.

  1. In the Samples tab > Import Sample screen, below the Browse field, click Click here to download a Microsoft™ Excel™ template file.
    Note:

    The template file contains default sample attributes as columns. If additional custom sample attributes have been configured in the software, these attributes need to be added as columns to the template file.

    1. In the template file, fill in data for each sample, one sample per row. Required attributes are flagged with an asterisk (*).

      • Sample ID*

      • Collection Date*

      • Sample Source

      • Sample Condition*

      • Gender*

      • Sample Type*

      • %Cellularity

      • %Necrosis

      • Reference Interval

      • Notes

    2. Click Save or Save As to save the file.
  2. Click Browse, then navigate to the saved file and click Open.
  3. Click Import.

    A sample import progress bar followed by an import report will display. If the import process fails, an error message will indicate the reason for failure (e.g., an invalid character was used).

    Note:

    A non-case sensitive check of all Sample IDs entered imported is performed to prevent duplication. If a non-unique Sample ID is found, import will fail. If the import file has a required header that is not recognized by the software as a sample attribute (e.g., if the header text is misspelled), import will fail.

  4. Click Manage Samples to return to the sample list. Successfully imported samples will be listed.
