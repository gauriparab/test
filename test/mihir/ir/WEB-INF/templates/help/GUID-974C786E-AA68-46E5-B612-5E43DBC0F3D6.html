View DNA barcodes

In the tab > References screen > DNA Barcodes subtab, you can view the barcode IDs and sequences for each barcode set in the software.

  1. Click on the name of a barcode set in the list.

    A new window will open showing the ID and sequence of each barcode in the set.

  2. Click Back to return to the list.
