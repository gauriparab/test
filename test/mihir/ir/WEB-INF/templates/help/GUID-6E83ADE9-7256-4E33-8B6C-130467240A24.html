Search Audit Records

Use the Search toolbar to search for an existing record:

  1. Limit the search to narrow the number of search results:
    1. Click to select a Start and End Date.
      Note:

      Search results will include all records created since the Start Date entered up to and including the current date if an End date is not entered.

    2. Select a Username from the drop-down list to limit the results to those actions performed by the selected user.
      Note:

      Search results will include records generated by all users if no selection is made.

    3. Select an Action (Edit, Create, Obsolete, Delete) from the drop-down list to limit the results to that action.
    4. Select a Data Object Name from the drop-down list to limit the results to records that only apply to that data object.
  2. Click Search.

    The record(s) matching the search parameters will be listed.

  3. Click Clear to return to the complete list of records.

The search function can search based on all three criteria individually, sequentially, or collectively.
