Create a report template

You can create custom report templates for use in generating result reports.

  1. In the Assay tab > Presets screen, click the Report Templates subtab.

    The existing report templates are listed on this page. To view the details of a template, click on the report name.

  2. Click Add New, and fill out the fields in the form.
    Note:

    Fields identified with a red asterisk (*) are required.

  3. Under Sample Details, select the attributes to display by selecting the checkbox next to each attribute. You can also drag and drop the additional attributes below the columns into the columns.
  4. Enter a description of the test, then enter any comments and click Save.

    The new report template will appear in Report Templates list.
