Assays and Planned Runs

The Assay tab contains tools for managing and creating assays and Planned Runs. There are also tools for importing assays, setting up assay presets (annotation sets, classification sets, filter sets, copy number baselines, and report templates), and installing assay templates.

Assays contain the settings and parameters for analyzing sequencing results and determining valid variant calls. Assays also define the kits and chips used in a run, and specify the threshold values for quality control and variant detection. Manager and Administrator-level users can create custom assays.

Assays also define the kits and chips used in a run, and specify the threshold values for quality control and variant detection. The software includes several pre-installed assays, and Manager and Administrator-level users can also create custom assays.

Planned Runs contain all the required sample and library information for template preparation and sequencing. Each Planned Run includes a Run Short Code in alphanumeric and barcode formats. By scanning or entering the code, the Planned Run's settings are transferred to the instruments, and each run and its associated samples, libraries, and assay are tracked from template preparation through sequencing, data analysis, and export.
