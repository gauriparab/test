Create a Generic Sequencing Assay

The Generic Sequencing application is used to create a basic sequencing assay, with no application-specific post-run analysis. You can select kits, chips, primer panels, plugins, and adjustable QC and read mapping parameters for this assay.

  1. In the Create Assay screen > Application page, click the Generic Sequencing Application tile. Select the version of the variant calling algorithm you are using from the dropdown list in the tile (e.g., IR44), then click Next.
  2. In the Reference page, select the Genome Reference from the dropdown list (hg19 is installed with the system). Then click Next.
  3. In the Reagent page, select the kits, chips, and panel you are using. Make sure they are appropriate for the instruments you are using. Click Next.
  4. In the Controls page, edit the settings for the controls. The available controls are dependent on your kit and panel selections (e.g., the Sequencing Control is available with Ion PGM™ Hi‑Q™ kits). See Quality control metrics for a description of these settings. Click Next when done.
  5. In the QC page, you can enter new run control parameter values in the fields. See QC report tab for a description of these parameters. Click Next when done.
  6. In the Parameters page, a description of each adjustable read mapping parameter is provided. Click on the Main and Advanced tabs to view different parameters. Edit the values in the fields within the defined ranges, then click Next.
  7. In the Plugins page, select the analysis plugins to run, then click Next.
    Note:

    The available plugins depend on what is installed and configured in your software.

  8. In the Save page, enter an assay name and optional assay description. Click Save.

    The new Assay will appear listed in the Manage Assays screen. The Status column will indicate that assay is in Draft status. The new Assay is available to View, Edit, Lock, or Delete. See Managing Assays.

Note:

For an Assay to be used in a planned run, it must first be locked.
