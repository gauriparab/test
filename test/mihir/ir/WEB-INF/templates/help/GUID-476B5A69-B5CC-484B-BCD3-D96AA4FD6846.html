Create a DNA and Fusion assay

The DNA and Fusion application is used to create an assay for detecting and analyzing somatic DNA variants and RNA fusion targets in a sample. You can select kits, chips, primer panels, panel and hotspot BED files, plugins, and adjustable QC and analysis parameters.

  1. In the Create Assay screen > Application page, click the DNA and Fusion tile. Select the version of the variant calling algorithm you are using from the dropdown list in the tile (e.g., IR44), then click Next.
  2. In the Panel page, select the following information and then click Next. For more information about the selections on this page, see Managing references.

    • Select the Genome Reference from the dropdown list (hg19 is installed with the system).

    • Select the BED file for the DNA primer panel you are using from the Panel dropdown list, and the file for the fusion panel from the Fusion panel dropdown list.

      Note:

      Each panel file is a tab-delimited BED file that defines the coordinates of the regions of interest to be amplified with the selected primer set.

    • Select the DNA hotspots BED file from the Hotspots dropdown list, and the RNA fusion target regions file from the Fusion Reference list.

      Note:

      Each file is a tab-delimited file that indicates the positions where a DNA variant or RNA fusion target is expected to occur. It also annotates these positions.

    • Optional: Select the Generate Test Report checkbox, then select a Reporting Gene List from the Genes and Mutations to Report dropdown list. This will generate a Test Report of those genes and mutations in the list. A link to the Test Report will be included in the Results List screen.

  3. In the Reagent page, select the kits and chips you are using. Make sure they are appropriate for the primer panel and instruments you are using. Click Next.
  4. In the Controls page, edit the settings for the controls. The available controls are dependent on your kit and panel selections (e.g., the Sequencing Control is available with Ion PGM™ Hi‑Q™ kits). See Quality control metrics for a description of these settings. Click Next when done.
  5. In the QC page, you can enter new quality control parameter values in the fields. See QC report tab and MAPD Copy Number QC metric for a description of these parameters.
    1. Click Library QC and Run QC to toggle between the parameters.
    2. Click within the desired field, then enter the new value. Click Next when done.
  6. In the Presets page, you can select appropriate sets from the Annotation Sets and Classification Sets dropdown lists. See Assay presets for instructions on loading presets into the software. Click Next.
  7. In the CNV page, the following options are available. Click Next when done.

    • Select Run CNV Algorithm and select a CNV baseline from the dropdown list to analyze your results against a CNV baseline. See Copy number baselines for instructions on importing or creating a CNV baseline.

    • If the CNV algorithm checkbox is selected, you can also select Apply copy number adjustment based on tissue content of each sample, for samples that meet the minimum cellularity percentage in the field below. Note that the Sample % Cellularity is entered when the sample was created in the software.

  8. In the Parameters page, a description of each adjustable parameter is provided. Click on the parameter categories (> Read Mapping, etc.,) and the Main and Advanced tabs to view different parameters. Edit the values in the fields within the defined ranges, then click Next.

    For a description of the variant finding parameters, see Variant finding parameters. For a description of the Fusions parameters, see Fusions parameters

  9. In the Plugins page, select the analysis plugins to run, then click Next.
    Note:

    The available plugins depend on what is installed and configured in your software.

  10. In the Save page, enter an assay name and optional assay description. Click Save.

    The new assay will appear listed in the Manage Assays screen. The Status column will indicate that assay is in Draft status. The new assay is available to View, Edit, Lock, or Delete. See Managing assays.

Note:

For an assay to be used in a planned run, it must first be locked.
