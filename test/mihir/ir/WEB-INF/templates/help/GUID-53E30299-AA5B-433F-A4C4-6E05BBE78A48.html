Allele Coverage tab

The Allele Coverage tab in the Results Report screen displays the following information for the locations defined by the hotspot file employed (incidental results are not reported):

Column

Description

Chromosome

The chromosome number of the reference.

Position

The one-base position of the hotspot in the reference.

Target ID

Name of the target region where the hotspot is located is AMPL by default. This value is not available (N/A) if no target regions were defined (i.e., the full genome was used).

Hotspot ID

The name of the hotspot as defined in the BED file.

Ref

The reference base of the hotspot.

Cov

The total reads covering the hotspot position.

A Reads

The number of reads with an A at the hotspot position.

C Reads

The number of reads with a C at the hotspot position.

G Reads

The number of reads with a G at the hotspot position.

T Reads

The number of reads with a T at the hotspot position.

Deletions

The number of reads with a deletion at the hotspot position.

+ Strand Cov

The total number of forward strand base reads at the hotspot position.

- Strand Cov

The total number of reverse strand base reads at the hotspot position.

Note:

Table does not include coverage for hotspot insertion variants.
