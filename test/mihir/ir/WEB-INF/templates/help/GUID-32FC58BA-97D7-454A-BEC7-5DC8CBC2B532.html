Manage Logs

Manage logs allows the Administrator to set retention times and allow for auto-deletion of logs.

  1. Click the Manage Log button.

    The Manage Log File dialog window will open.

  2. From the Retention Period drop-down list select the number of months logs are to be retained on the server.
  3. Select Enable Auto Deletion to automatically delete log files after the designated retention period.
  4. Click Save.
