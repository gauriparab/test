Display message: Instrument temperature too low

  • Room temperature is below 20°C

  • Hardware issue

Note:

The data created during a run with this alarm raised may still be used if all the QC metrics are met.

If the ambient room temperature is below 20°C, raise it. If the problem persists, contact Technical Support (see c_Obtaining support_Ion).
