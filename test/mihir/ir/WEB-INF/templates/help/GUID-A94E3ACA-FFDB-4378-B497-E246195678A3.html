Disable the Smart Monitoring service

The Smart Monitoring service allows company personnel to monitor the status of the Ion PGM™ Dx System remotely through an internet connection. Smart Monitoring employs multiple layers of security, including a Secure Sockets Layer (SSL) and Lightweight Directory Access Protocol (LDAP) authentication, to provide real-time troubleshooting and problem resolution for the Ion PGM™ Dx System. For a detailed description of the Smart Monitoring Service, see the Smart Monitoring Service Product Bulletin: Leveraging the power of the Internet while maintaining system security (Part no. 121PB07-03).

The Smart Monitoring service is active by default and can be disabled if required.

  1. Below the Smart Monitoring Service Details header, click Stop.

    A confirmation dialog window will open, “ Are you sure you want to Stop Smart Monitoring service?”

  2. Click Yes to disable the service.

    The Smart Monitoring service dialog will collapse and the option Start will replace Stop.

  3. Click No to return to the Services page.
  4. To reactivate the Smart Monitoring service click Start.

    A confirmation dialog window will open, “ Are you sure you want to Start Smart Monitoring service?”

  5. Click Yes to confirm restarting the Smart Monitoring service function.
