Set security policies

Basic user account security policies (e.g., number of permissible failed login attempts, password lifetime, duration of inactivity before session times out) are defined within this dialog window. To make changes to these basic policies:

  1. Click Policies.

    The Policies dialog window will pop-up.

  2. Select the appropriate value(s) from the drop-down lists to set the user account Suspension, Password, and Session policies.
  3. Click the Enabled checkbox to enable the Session policies.
  4. Click Save.
