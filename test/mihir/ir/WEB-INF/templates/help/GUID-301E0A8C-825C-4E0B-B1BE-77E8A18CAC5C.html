View the Results List

In the Data tab > Completed Runs & Results screen, samples that have been sequenced and analyzed are listed by Sample ID. The number of results for each sample is listed in the Results column.

Note:

There can more than one result per sample if a sample has been re-sequenced using a new Planned Run, or if the sample was reanalyzed using a Reanalysis assay.

  1. Click the number in the Results column to open the Results List screen for the Sample ID.
  2. In the screen, each Planned Run used to generate results for the sample is listed. The status of the run is shown in the Run Status column.
    Note:

    Results generated by reanalysis are indicated in the Assay column with the reanalysis assay name and the symbol .

    • Under the Planned Run column, click Audit to display the list of users who created/edited the Planned Run. You can export and print information from the list from the Audit Trail dialog box.

    • Click CSA (Customer Support Archive) to download all the log files from the instruments used in the run. CSA log files may be useful when troubleshooting issues with the instruments.

    • Click the Planned Run name to view the information entered when the Planned Run was created.

    • Click the assay name to view the assay details

    • Click Lab Report to download the lab report.

    • Click View Result to open the Result Report screen.

    • Click Reanalyze to reanalyze the results using a Reanalysis assay.

      Note:

      • This button is only available if a Manager or Administrator has created a Reanalysis assay. See Create a Reanalysis Assay.

      • The sequencing data from the entire run will be reanalyzed, resulting in new Results Reports for all the samples in a run, not just the individual sample of interest.

    • Click Refresh to update the list (e.g., status of the run in the Run Status column).
