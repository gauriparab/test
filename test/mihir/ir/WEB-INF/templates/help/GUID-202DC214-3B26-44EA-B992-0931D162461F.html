Manage panel files

In the tab > References screen > Panels subtab, you can view the panel BED files available in the software and add new panel files. See Pre-installed panels for a list of primer panels supported by the pre-installed BED files in the software.

  • To add a new panel file, click Add New and enter a name, select a genome reference, and select the BED file containing the panel information. Click Save when done.
    Note:

    Extremely large files take time to upload and may not appear immediately.

  • To export a panel file, click on Export in the Actions column.
  • To obsolete a panel file (making it unavailable in the software), click on Obsolete in the Actions column. Pre-installed panel files cannot be obsoleted.
