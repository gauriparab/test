Manage fusion panels

In the tab > References screen > Fusion Panel subtab, you can view the fusion panel BED files available in the software and add new panel files. See Pre-installed panels for a list of kits supported by the pre-installed BED files in the software.

  • To add a new fusion panel file, click Add New and enter a name, select a panel reference, and select the BED file containing the panel information. Click Save when done.
    Note:

    Extremely large files take time to upload and may not appear immediately.

  • To export a fusion panel file, click on Export in the Actions column.
  • To obsolete a fusion panel file (making it unavailable in the software), click on Obsolete in the Actions column. Pre-installed panel files cannot be obsoleted.
