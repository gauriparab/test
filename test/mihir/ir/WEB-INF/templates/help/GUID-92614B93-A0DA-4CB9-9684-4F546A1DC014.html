Create a DNA Germline Default Assay

The DNA Germline Default application is used to create a single-sample DNA germline assay that uses Ion PGM™ Dx System kits and chips. You can select primer panels, panel and hotspot BED files, plugins, and adjustable QC and analysis parameters for germline variant calling.

Note:

For a description of the default QC and parameter values in this assay, see DNA Germline Default Assay settings.

  1. In the Create Assay screen > Application page, click the DNA Germline Default tile. Select the version of the variant calling algorithm you are using from the dropdown list in the tile (e.g., IR36), then click Next.
  2. In the Panel page, select the following information and then click Next. For more information about the selections on this page, see Managing references.

    • Select the Genome Reference from the dropdown list (hg19 is installed with the system).

    • Select the BED file for the primer panel you are using from the Panel dropdown list.

      Note:

      The panel file is a tab-delimited BED file that defines the coordinates of the regions of interest to be amplified with the selected primer set.

    • Select the hotspots BED file from the Hotspots dropdown list.

      Note:

      The hotspot file is a tab-delimited file that indicates the positions where a variant is expected to occur within your genotyping assay. It also annotates these positions with the reference allele, variant allele, and anchor base.

  3. In the Reagent page, the Ion PGM™ Dx System kits and chips are preselected. Click Next.
  4. In the Controls page, you can edit the values for the Ion Dx Templating Control. See Quality control metrics for a description of these settings. To restore the default values, click Reset All. Click Next when done.
  5. In the QC page, you can enter new quality control parameter values in the fields. See QC report tab for a description of these parameters.
    Note:

    The default values are set to the minimum allowed for this assay type. You can only increase the stringency of the parameters.

    1. Click Library QC and Run QC to toggle between the parameters.
    2. Click within the desired field, then enter the new value. Click Reset All to reset all parameters to minimum requirements.
    3. Click Next.
  6. In the Parameters page, use the scroll bar to view the variant finding parameters. See Variant finding parameters for a description. You can enter new (more stringent) values in the fields. Click Reset All to reset all parameters to the default minimum requirements.
    Note:

    Minimum read depth for germline claims is set to 6 reads, 3 reads of each strand.

  7. In the Plugins page, select the analysis plugins to run, then click Next.
    Note:

    The available plugins depend on what is installed and configured in your software.

  8. In the Save page, enter an assay name. You can also enter an assay description. Click Save.

    The new Assay will appear listed in the Manage Assays screen. The Status column will indicate that assay is in Draft status. The new Assay is available to View, Edit, Lock, or Delete. See Managing Assays.

Note:

For an Assay to be used in a planned run, it must first be locked.
