Create a new assay (Manager/Administrator only)

Manager and Administrator level users can create new assays using the template wizard.

  1. Under the Assay tab, click Create Assay . (Alternatively, click Add New in the Manage Assays screen.)
  2. The Create Assay screen has a list of steps across the top, and each step is a new page with different selections and settings.

    • The active step highlighted in blue.

    • Use the Next and Previous buttons on each page to navigate the steps.

    • Required fields within each page are indicated with an asterisk (*).

    • The Summary pane to the right displays a list of your selections and settings as you move through the steps.

  3. To begin, select from the list of applications on the screen, which are described in the links below.
  • Create a DNA Germline Default Assay
  • Create a DNA Germline assay
  • Create a DNA and Fusion assay
  • Create a DNA Somatic Assay
  • Create a RNA Fusion Assay
  • Create a 16S Metagenomics assay
  • Create a Generic Sequencing Assay
