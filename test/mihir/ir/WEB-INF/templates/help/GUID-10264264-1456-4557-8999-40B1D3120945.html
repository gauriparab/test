Execute a Planned Run

Planned runs are listed in the Assay tab >  Planned Runs screen. Runs that are ready to be performed have the Execute command available in the Actions column.

Executing a Planned Run sends the run to the Ion OneTouch™ Dx Instrument or Ion Chef™ Instrument, but does not initiate the run. It creates a date time stamp in the the software indicating when the run was sent. Additionally, the system checks that all required information has been entered.

The Planned Run is initiated on the template preparation and sequencing instruments upon entering the Run Short Code or scanning the Planned Run barcode into the respective instruments.

In the Assay tab > Planned Runs screen:

  1. Click the To Be Started button to limit the list of Planned Runs to only those ready to be executed.
  2. Locate the Planned Run in the list, then under the Actions header click Execute.
    Note:

    If the Ion PGM™ Torrent Server has ≤10% free disk space, you will be alerted to clear disk space before the run can be executed.

    The Execute Planned Run dialog will open.

  3. In the Tube Label field:

    • For template preparation using the Ion Chef™ Instrument, click inside the field and scan the barcode on the Ion Chef™ Library Sample Tube. The barcode number will populate the field, and will be tracked by the instrument.

    • For template preparation using an Ion OneTouch™ instrument, enter the text that will be used to label the library tubes during the run. This can be any combination of letters and numbers. This label text is tracked by Ion PGM™ Dx System instruments, but not by non-Ion PGM™ Dx System instruments. Note that the software will not allow use of the same Tube Label text within 7 days.

  4. Click inside the Template Prep Kit barcode field and scan or enter the barcode from the Ion OneTouch™ Dx Template Reagents Kit box.
    Note:

    If you are using the Ion Chef™ Instrument, skip this step.

    Note:

    Be sure to scan or enter the barcode from the actual Ion OneTouch™ Dx Template Reagents Kit box that will be used in the run.

  5. Click Save to save your changes.
  6. The Review Planned Run dialog will open. Write down the Run Short Code and/or click the Print button to print the scannable barcode. The code must be entered or scanned at each instrument for tracking and verification.
  7. Click Close to close the dialog and execute the run. Then proceed to combining libraries and cleaning the Ion OneTouch™ Dx Instrument or Ion Chef™ Instrument prior to initiating the run on the instrument.
Note:

The executed Planned Run will remain accessible on the Monitor tab for up to 72 hours.
