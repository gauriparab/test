Managing sample attributes (Manager/Administrator)

Torrent Suite™ Assay Development Software has the following predefined sample attributes, which are listed in the Samples tab > Manage Attributes screen:

  • Sample ID*

  • Sample Source

  • Sample Condition*

  • Sample Type*

  • Collection Date*

  • Receive Time*

  • Gender*

  • Reference Interval

  • %Cellularity

  • %Necrosis

  • Notes

* – indicates a field required during sample creation; however, all fields except Sample Name and Description, are required for sample use in a Planned Run

Predefined attributes are locked and cannot be edited. You can create and manage custom sample attributes using the tools in the Manage Attributes screen.

Note:

LIMS users must create custom attributes before importing sample and Planned Run information from LIMS for the attributes to be propagated through to Ion PGM™ Torrent Server output files. The Torrent Suite™ Assay Development Software will ignore all input file content that is not a recognized attribute.
