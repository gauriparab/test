Create a 16S Metagenomics assay

The 16S Metagenomics application is used to create an assay for determining population diversity in polymicrobial samples via detection of 16S rRNA gene variable regions. You can select kits, chips, primers, plugins, and adjustable QC and analysis parameters for this assay.

  1. In the Create Assay screen > Application page, click the 16S Metagenomics tile. Select the version of the variant calling algorithm you are using from the dropdown list in the tile (e.g., IR44), then click Next.
  2. In the Reference page, select one or both of the following pre-installed references. To select a single reference, click on it. To select both references, hold down the Shift or Ctrl button and click on both. Then click Next.

    Reference

    Description

    Curate MicroSEQ™ 16s Reference Library v2013.1

    Library that contains a list of reference 16S rRNA sequences from bacteria against which an unknown bacterial sequence can be compared.

    Curated Greengenes v13.5

    Library of more than 1.2 million references that contain taxonomies at the family level and above.

    If both references are selected, data is first mapped against the MicroSEQ™ reference, then the remaining reads are mapped against the Greengenes reference.

  3. In the Reagent page, select the kits, chips, and panel you are using. Make sure they are appropriate for the instruments you are using. Click Next.
  4. In the Controls page, edit the settings for the controls. The available controls are dependent on your kit and panel selections (e.g., the Sequencing Control is available with Ion PGM™ Hi‑Q™ kits). See Quality control metrics for a description of these settings. Click Next when done.
  5. In the QC page, you can enter new quality control parameter values in the fields. See QC report tab for a description of these parameters.
  6. In the Primers page, select No Primers, Default Primer (provided with the Ion 16S™ Metagenomics Kit), or Custom Primer, and click Next.
    Note:

    For custom primers, you must first upload a FASTA file containing the primer sequences. See Create and view primers.

    Note:

    When the primer information is missing, no trimming is performed on your reads. A warning message appears during analysis review when primer information is missing.

  7. In the Parameters page, a description of each parameter is provided. The default settings are optimized for metagenomics assays. Edit the values in the fields within the defined ranges, then click Next.
  8. In the Plugins page, select the analysis plugins to run, then click Next.
    Note:

    The available plugins depend on what is installed and configured in your software.

  9. In the Save page, enter an assay name and optional assay description. Click Save.

    The new assay will appear listed in the Manage Assays screen. The Status column will indicate that assay is in Draft status. The new assay is available to View, Edit, Lock, or Delete. See Managing assays.

Note:

For an assay to be used in a planned run, it must first be locked.
