Display message: Software versions incompatible. Go to the Options menu and update the software.

The system software was updated, but the instrument software was not.

After the system software has been updated, update the instrument software as follows:

  1. On the main menu of the instrument, press Options and follow the instructions to check for and install updates.

  2. When installation is complete, follow the onscreen prompts to reboot the instrument.

    Note:

    You must reboot the instrument before proceeding.
