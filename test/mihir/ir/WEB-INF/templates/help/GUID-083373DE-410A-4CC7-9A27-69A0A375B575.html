Create a Filter Chain

Filter chains are a set of filters that you apply to variants identified in your assay. To create a filter chain:

  1. In the Assay tab > Presets screen > Filter Chains subtab, click Add New.

    The Create Filter Chain window appears.

  2. Enter a name for your filter chain.
  3. Select the filter you want to use from the dropdown list.
    Note:

    If the dropdown list does not populate with the filters available in the software, you may need to reboot the Ion PGM™ Torrent Server.

  4. Select variants to filter in. Click Set. The filters contained in the filter chain are now shown on the right side of the window.
  5. (Optional) add a description.
  6. Click Save.

    A confirmation message appears, and your new filter chain is listed in the screen and is available for selection.
