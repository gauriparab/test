Torrent Suite™ Assay Development Software Help

Welcome to the Torrent Suite™ Assay Development Software Help System

This help system provides instructions for using Torrent Suite™ Assay Development Software to:

  • Create and manage samples, libraries, assays, and Planned Runs on the Ion PGM™ Torrent Server

  • Execute and monitor the status of runs on the Ion PGM™ Dx System instruments, as well as the Ion Chef™ Instrument

  • View data, results, and QC reports after a run

  • Download data files for all the results from a run

  • Export and print audit trails of samples and runs

  • Create and manage annotation and classification sets, genome references, report templates, baselines, primer sets, plugins, and other reference files

  • Create and manage user accounts

The Torrent Suite™ Assay Development Software home page depicts the four-step process whereby samples are processed. Position the cursor over each step on the page for a brief description.

  • Sample — Create and manage sample records and libraries

  • Plan — Plan a sequencing run by selecting libraries and assay parameters

  • Monitor — View the progress of the run in real time and assess the run metrics as they are generated

  • Data — View run results and sign off on reports

  • Switch software mode (administrator only)
