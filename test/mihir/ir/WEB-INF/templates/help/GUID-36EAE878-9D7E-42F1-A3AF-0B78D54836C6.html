Control Fragments

Internal controls are pre-defined and cannot be modified by the user. Listed in the Control Fragments pane are:

Column

Description

Name

the control fragment name

Sequence

the single letter nucleotide sequence of the control fragment

Control

the process in which the control fragment is used

Status

indicates that the control fragment information is locked and cannot be modified by the user
