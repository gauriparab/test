Display message: Instrument temperature too high

Note: The data created during a run with this alarm raised may still be used if all the QC metrics are met.

  • Room temperature is too high

  • Clogged filter or blocked airway on the instrument

  • Hardware issue (fan is not running or running too slowly)

  1. If the ambient room temperature is above 30°C, lower it.

  2. Make sure that the round filter on the back panel of the instrument has unrestricted airflow. If the filter is clogged with dust, clean it as follows:

    1. Pinch the dirty filter with your fingers, then remove it from the instrument.

    2. Shake the filter over a waste container to remove most of the dust.

    3. Rinse the filter with running water to remove any remaining dust. The water flow should be from the inside-facing surface to the outside-facing surface through the filter.

    4. Air dry the filter.

    5. Blot any remaining dust from the filter using tape.

    6. Reinsert the filter.

If the problem persists, contact Technical Support (see c_Obtaining support_Ion).
