Managing Planned Runs

The Assay tab > Planned Runs screen displays a list of all Planned Runs with the most recently created run on top. In this screen, a user can:

  • Create Planned Runs

  • Execute Planned Runs

  • Sort and search the list of Planned Runs

  • View the audit trail of a Planned Run

  • Review a Planned Run

  • Edit a Planned Run

  • Delete a Planned Run (Manager/Administrator level users only)

  • Review/Print a Planned Run
  • Search and Sort Planned Runs
  • Review a Planned Run audit trail

  • Delete a Planned Run (Manager/Administrator only)
