Add a new user

Note:

To allow sending email notifications to new user's, the Ion PGM™ Torrent Server email settings must be configured before adding new user accounts. See Configure Email Settings for more information.

To create a new user account:

  1. Click the Add New button.

    The Create New User dialog will open.

  2. Enter the new:
    Note:

    Required fields are indicated by a red asterisk (*).

    • User Name

    • Account Information

      • First Name

      • Last Name

      • Email Address

      • Comments

  3. Click the Electronic Signature checkbox to enable the ability to sign off on reports.
  4. Select from the dropdown list the user access level (Role): Administrator, Manager, or Operator.
  5. Click Save. The Ion PGM™ Torrent Server will send a temporary password to the email address entered in new user's account.
