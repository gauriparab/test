Edit an assay

When a Manager or Administrator first creates an Assay, it will be listed as "Draft" in the Status column in the Manage Assays screen. While the Assay is in draft status, you can edit it.

  1. Click on Edit below the Assay name in the Manage Assays list. The software will reopen the Create Assay workflow.
  2. Edit the parameters on each page as desired, then click on Save when done.
