Torrent Suite™ Assay Development Software Help

I created a Planned Run but the Ion PGM™ Dx Sequencer does not recognize the Short Code

Planned Run was not executed.

In the Torrent Suite™ Assay Development Browser:

  1. Select the Planned Run, then click Execute.

  2. Fill out the information in the dialog, then click Save.

The Ion PGM™ Dx Sequencer and Ion PGM™ Torrent Server have lost connection.

Check the network connection to the Ion PGM™ Torrent Server to make sure the connection is established, then reboot the instrument.
