Create and view annotation sets

The software provides annotation content from a wide range of public resources and provides the ability to import and leverage custom annotations.

In the Assay tab > Presets screen, click on the Annotation Sets subtab. The existing annotation sets are listed on this page.

Click on a set name to view the details of that set.

To add a new annotation set:

  1. Click Add New.
  2. Enter a name for the set.
  3. Optional: Enter a description.
  4. Select the annotation source from the Choose Type dropdown list.

    • 5000 Exomes - Population frequency information from the 5000 exomes project. Source: http://evs.gs.washington.du/EVS/

    • ClinVar - Assessment of impact of the variant observed from NCBI ClinVar database. Source: http://www.ncbi.nlm.nih.gov/clinvar/

    • COSMIC - Catalog of known somatic mutations in cancer. Source: http://cancer.sanger.ac.uk/cancergenome/projects/cosmic

    • dbSNP - Database of genomic variants. Source: http://www.nci.nlm.nih.gov/projects/SNP

    • DGV - Database of Genomic Variants: a curated database of human genomic structural variation. Source: http://dgv.tcag.ca/dgv/app/home

    • DrugBank - List of drugs known to target the gene(s) affected by the variant. Source: http://drugbank.ca

    • Gene Model - Set of genes the variant overlaps. Sources: http://www.ncbi.nih.gov/refseq/ and http://www.ensembl.org/index.html

    • Gene Ontology - Standardized ontology for gene and gene projects (e.g. functional role or localization). Source: http://geneontology.org

    • Gene Set (Custom) - Focus on variants found within specific set of genes

    • Genomic Regions (Custom) - Focus on variant found within specific gene regions

    • OMIM - Online Mendelian Inheritance of Man. Source: http:www.ncbi.nih.gov/omim

    • Pfam - Protein domain families in the coded protein. Source: http://pfam.sanger.ac.uk

    • PhyloP Scores - Measure of conservation of the protein across a wide range of organisms. Source: http://compgen.bscb.cornell.edu/phast/

    • SIFT / PolyPhen - Prediction of the functional effect of a variant on a protein. Source: http://sift.jcvi.org

    • Transcript Set (Custom) - Preferred transcripts used to determine coding regions of genes. Sources: http://www.ncbi.nlm.nih.gov/refseq and http://www.ensembl.org/index.html

    • Variant DB (Custom) - Annotation source preset used to annotate analyses.

  5. The available versions of the source will be listed. Select the Use button next to the desired version. The source will be listed under Selected Sources.
  6. Repeat this process for each source you want to add to the set.
  7. Click Save to save the set.

    The annotation set will listed on the page and will be available when you create an assay.
