QC report tab

The QC Report tab in the Results report window displays the following information for both the and the internal controls. This information is also accessible through the Monitor tab within 72 hours of starting the run on the Ion OneTouch™ Dx Instrument or Ion Chef™ Instrument.

Metric

Description

Run QC Evaluation Metrics

Key Signal

Average peak signal from incorporation trace measured across the 1-mer incorporations in all library sample key sequences.

Percent Loading

Percentage of chip wells that contain an ISP. (The percentage value considers only potentially addressable wells.)

Metric

Description

Library QC Evaluation Metrics

Mappable Fusion Reads

The number of reads that are mapped to the fusion reference file. Applies to RNA libraries only.

Mean AQ20 Read Length (bp)

Average length, in base pairs, at which the error rate is ≤1% for all aligned reads of a sample.

Mean Read Length (bp)

Average length, in base pairs, of all reads reported for a given barcode.

Percent Reads

Percentage of all usable library reads that contain the Barcode ID assigned to a given sample divided by total addressable wells.

Metric

Description

Control QC Evaluation Metrics[1]

Base Call Accuracy

1–(total number of errors for all positions in the control / total number of aligned bases) for Control Fragment-1 (CF-1) reads.

Key Signal

Average peak signal from incorporation trace measured across the 1-mer incorporations in the control fragment key sequences.

Mean AQ20 Read Length (bp)

Average length, in base pairs, at which the error rate is ≤1% for all aligned reads of CF-1. This metric assesses the read accuracy.

Percent Reads

Percentage of CF-1 reads normalized by the total addressable wells in a run. This metric assesses the CF-1 coverage.

Histogram of Read Length [2]

A library specific Read Length Histogram is provided.

Metric

Description

Sign Off Details[3]

Sign Off Comments

Note indicating that the report is signed off.

Meaning of the signature

Approval by electronic signature.
