Sort samples

In the Samples tab > Manage Samples screen, the list of samples is displayed with the most recently created sample on top by default. To return to the default display click Manage Samples.

  • To list only those samples which have not been extracted, click To Be Extracted.
  • To list only those samples which have not been prepared as a library, click To Be Prepared.
  • To list all samples, click All.
  • To sort the list:
    1. Click on a column header to sort the list by the entries in that column.
    2. Click on the column header again to reverse the order.
    3. Click Manage Samples to return to the default order (most recently created on top).
