Torrent Suite™ Assay Development Software Help

Samples and Libraries

Prior to sample and library preparation, sample and library information must be entered into the Torrent Suite™ Assay Development Software. This information is then tracked by the software throughout the entire system workflow.

In the Samples tab, you can:

  • Create and manage samples and their attributes

  • Prepare libraries and library batches

  • Create Planned Runs

  • Add a new sample
  • Managing Samples
  • Enter the extraction kit barcode
  • Import samples
  • Managing sample attributes (Manager/Administrator)
  • Prepare a library batch
  • Manage libraries
