Update the Archive Settings

To maintain sufficient free disk space we recommend implementing a systematic plan to archive the data from the Ion PGM™ Torrent Server. Archiving the data transfers all intermediate run data and final results files to an external location not on the Ion PGM™ Torrent Server freeing disk space.

Note:

System administrators of your Torrent Suite™ Assay Development Software are responsible for configuring and maintaining a data archive.

Administrator level users will receive disk-full notification emails when the Ion PGM™ Torrent Server has ≤10% free disk space remaining.

Note:

To receive disk-full notifications a valid email address must be on file, see User Management (Administrator). Additionally the Ion PGM™ Torrent Server must be configured to allow sending emails, go to "Gear" tab > Configuration > Email Settings to configure the server. For more information, see Configure Email Settings.

To update the Archive Settings:

  1. From the Auto archive after drop-down list, select the number of months before data from a run is archived.
  2. Click in the Archive Directory field and enter the file path.
  3. Click Save.
To free up additional disk space:

  • Delete incomplete runs.

  • Reduce the Auto archive after time.
