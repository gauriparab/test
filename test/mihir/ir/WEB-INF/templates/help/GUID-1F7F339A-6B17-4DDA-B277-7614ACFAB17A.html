Instrument View

The instruments connected to the Ion PGM™ Torrent Server are individually listed in the Instrument View screen under the Monitor tab. To update the information in the screen during a run or following completion of a run, click the Refresh button.

The following information is displayed in the screen:

Column

Description

State

Function the instrument is currently performing (e.g., cleaning, initializing, running etc).

Status

The current status of the active run function, including data analysis. Allows remote monitoring of instruments.

Last Water Clean

Date and time the last water cleaning of the instrument was performed.

Last Chlorite Clean

Date and time the last chlorite cleaning of the instrument was performed.

Serial

Ion PGM™ Dx Sequencer or Ion OneTouch™ Dx Instrument or Ion Chef™ Instrument serial number.

PQ Status

Indicates whether the PQ status of the instrument is Verified or Expired. Click the Download Report link to download and view the PQ result report.

Next PQ Due

Indicates the expiration date of the current PQ verification run.

Note:

For the Ion OneTouch™ Dx Instrument, Last Clean refers to the pre-run cleaning and is not updated following the post-run cleaning.
