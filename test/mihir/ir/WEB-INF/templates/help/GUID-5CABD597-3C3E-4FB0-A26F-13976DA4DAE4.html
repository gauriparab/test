About page

The About page contains links to webpages where the user can obtain more information and assistance.

  • Support Contact Information– a link directing users to the thermofisher.com/technical-resources website where they can access technical support and product information.

  • Instrument Diagnostics – contains the initialization logs of all Ion PGM™ Dx Sequencer instruments connected to the Ion PGM™ Torrent Server and allows the user to:

    • download the diagnostic archive file (in .zip format)

    • view the Init.log file

    • download the Installation Acceptance Report (.pdf format)

  • End User License Agreement – opens the End User License Agreement (EULA).
