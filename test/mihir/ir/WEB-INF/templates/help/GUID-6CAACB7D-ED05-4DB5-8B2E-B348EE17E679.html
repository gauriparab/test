Manage primers

In the tab > References screen > Primers subtab, you can view the primer sequences in the software.

Custom primers may be added in the form of FASTA files. When entering primers, the names of forward primers must end in "_f*" and reverse primer names must end in "_r*". Primers in a pair must otherwise have identical names, so the software can match them during the analysis. An example of a FASTA formatted file is provided below: 

>tc_TEST1_for
ACGCGGTCCAACAGCT
>tc_TEST1_rev
TTACCGGGCGTGCCGG
>tc_TEST2_for
CCAACTCGGTCTACT
>tc_TEST2_rev
GCGTATGCACCGGG
Note:

Primer sequences that are selected when creating an assay are trimmed from the reads prior to mapping.

To upload a FASTA file containing your primer sequences:

  1. Click the Add New button.
  2. Enter a name for the primer and click Select File. Then click Save.
  3. The primer will appear listed in the screen.
