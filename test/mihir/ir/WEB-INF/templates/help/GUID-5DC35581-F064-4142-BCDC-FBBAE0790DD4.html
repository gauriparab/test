Network Settings

In the Configuration window the Network Settings tab is the default view. Information about the Ion PGM™ Torrent Server is displayed in the Network Settings tab. In the event of a problem with the Ion PGM™ Torrent Server the information displayed here can be used to troubleshoot the cause.

The Audit Trail of the Ion PGM™ Torrent Server Network Settings is available to view, export, and print by clicking on Audit Trail. For more information see Audit Records (Administrator).
