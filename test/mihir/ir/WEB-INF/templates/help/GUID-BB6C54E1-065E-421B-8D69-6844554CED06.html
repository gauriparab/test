Edit a Planned Run

You can edit Planned Runs that have not been executed yet.

  1. In the Assay tab >  Planned Runs screen, click Edit below the name of the Planned Run.
  2. The Edit Planned Run dialog will open. Change the settings in the dialog and click Save.
