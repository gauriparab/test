Create and view classification sets

Classification sets are user-defined sets for classifying variants detected by an assay.

  1. In the Assay tab > Presets screen, click on the Classification Sets subtab.

    The existing classification sets are listed on this page, and the terms in each set are listed in the Terms column.

  2. Click Add New. In the Create Classification List dialog, enter a name and a user-defined, comma-separated list of classification terms.
  3. Enter optional notes and click Save.

    The classification set will listed on the page and will be available when you create an Assay.
