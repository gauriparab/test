Delete a Planned Run (Manager/Administrator only)

  1. In the Assay tab >  Planned Runs screen, select the Planned Run(s) to be deleted by clicking in the check box adjacent to the Planned Run. Select all the Planned Runs on the page by selecting the check box above the column.
  2. Click (Delete).

    The Confirm Delete dialog will open with the message "Are you sure you want to delete the selected planned run(s)?

  3. Click Yes to delete the Planned Run(s), or click No to return without deleting.
