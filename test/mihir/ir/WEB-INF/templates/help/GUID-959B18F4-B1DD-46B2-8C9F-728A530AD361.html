View quality control result details

In the Monitor tab > Run View screen, the quality control details for each library in a Planned Run can be viewed for 72 hours after starting the Ion OneTouch™ Dx Instrument or Ion Chef™ Instrument run. To determine whether a library passed or failed quality control:

  1. Select the Planned Run and library from the respective dropdown lists. In the Sequencing pane, check that the Analysis Status reads Completed.
  2. Click View QC Details to view all quality control metrics for the run.
  3. Under the QC Status column, check that all Run QC, Library QC, and Control QC metrics passed.

    Within a given sequencing run, a library may produce a Passed or Failed result as long as the internal control fragment and run quality control metrics pass. A failed Run QC metric or failed Ion Dx CF-1 Control fragment QC metric invalidates all library results. All passing QC metrics are listed in green; failures are indicated in red.

  4. View QC details for additional libraries sequenced as part of the same Run by selecting the library of interest from the dropdown list.

    The QC details for the selected library will be displayed.

  5. Click to collapse the QC details view.
