Planned Run Validations

In order for the Create Plan API request to be valid, the following conditions must be met:

  • Planned Run name must be unique within Torrent Suite™ Assay Development Software database.

  • LibraryPrepID's must exist within the Torrent Suite™ Assay Development Software database.

  • Tube labels must be unique among all Planned Runs created or executed in the past 7 days. Tube label must be 10 characters or less in length, and only contain alphanumeric characters (0–9 and A to Z), full stop/period (.), underscore (_), and hyphen (-). Tube labels are case insensitive.
