Review/Print a Planned Run

You can review and print the information in a Planned Run when you execute it. Alternatively:

  1. In the Assay tab >  Planned Runs screen, click the Planned Run name in the list.
  2. In the View Planned Run dialog, review the information entered or generated when you created the run, and click on Print to print it.
