Search and Sort Planned Runs

The list of Planned Runs is displayed in the Assay tab >  Planned Runs screen with the most recently created on top by default. To return to the default display, click Planned Runs.

  • To search the Planned Run list:
    1. Enter the full, or partial, Planned Run name into the Planned Run Name field.
    2. Click Search.

      The Planned Run(s) matching the search parameters will be listed.

    3. Click Clear to return to the complete list of Planned Runs.
  • To sort the Planned Runs list:
    1. Click on the header name of interest.

      The list of Assays will reorder alphabetically based upon the header name selected. Run Short Code, Planned Run Name, Assay, and Tube Label.

    2. Click on the header name a second time to reverse the order of samples displayed.
  • Click To Be Started to limited the list of Planned Runs to only those runs that are ready to be executed.
