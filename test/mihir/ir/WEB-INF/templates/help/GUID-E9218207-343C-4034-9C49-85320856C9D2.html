Assays

Assays contain the settings and parameters for analyzing sequencing results and determining valid variant calls. Assays also define the kits and chips used in a run, and specify the threshold values for quality control and variant detection. The software includes several pre-installed assays, and Manager and Administrator-level users can also create custom assays.
