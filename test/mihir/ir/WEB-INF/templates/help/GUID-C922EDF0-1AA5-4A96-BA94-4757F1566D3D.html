Manage Reporting Gene Lists

A Reporting Gene List is a list of selected genes and mutations used in generating Test Reports for a particular assay. When creating an assay, you can select the Reporting Gene List to use to generate the report. The tools for managing Reporting Gene Lists are located in the tab > References screen > Reporting Gene List subtab.

  • To create a new list, click Add New, then enter a name for the list and select the file containing the gene information.
    Note:

    The file can be a .txt, .csv, or Microsoft™ Excel™ file with the following mandatory columns: Gene Name, Mutation ID, and Display Name. The Test Report will include only the variants matching the mutation ID for the corresponding gene. The file can also include up to six custom columns. The file name should have no spaces or special characters.

  • To a view the contents of an existing Reporting Gene List:
    1. Click the name of the list in the Name column.

      A new screen will open showing the gene names, mutation IDs, and other information about the genes in the list.

    2. Click Back to return to the list.
  • To export a list, click Export in the Actions column.
  • To obsolete a list, making it unavailable in the software, click Obsolete.
    Note:

    Existing Test Reports will continue to display the genes in the obsoleted list, but users will be unable to generate new Test Reports using the list.
