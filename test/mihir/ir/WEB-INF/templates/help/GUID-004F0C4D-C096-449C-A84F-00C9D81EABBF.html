Manage fusion references

The tab > References screen > Fusion Reference subtab lists the fusion reference files available in the software.

  • To add a new fusion reference file, click Add New and enter a name and select the FASTA file containing the reference information. Click Save when done.
    Note:

    Extremely large files take time to upload and may not appear immediately.

  • To export a fusion reference file, click on Export in the Actions column.
  • To obsolete a fusion reference file (making it unavailable in the software), click on Obsolete in the Actions column. Pre-installed files cannot be obsoleted.
