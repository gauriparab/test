Quality control metrics

In the Run View screen, once analysis of a sequencing run is complete, the following quality control details can be viewed.

QC detail

QC metric…

Run QC

  • Key Signal: average peak signal from incorporation trace measured across the 1-mer incorporations in the library key sequences.

  • Percent Loading: percentage of chip wells that contain an ISP (The percentage value considers only potentially addressable wells).

Library QC: DNA

  • Mean AQ20 Read Length (bp): average length, in base pairs, at which the error rate is ≤1% for all aligned reads of a library.

  • Mean Read Length (bp): average length, in base pairs, of all reads reported for a given library.

  • Percent Reads: number of library reads normalized by the total addressable wells in a run.

Control QC: CF-1

  • Base Call Accuracy: 1–(total number of errors for all positions in the control / total number of aligned bases) for Control Fragment-1 (CF-1) reads.

  • Key Signal: average peak signal from incorporation trace measured across the 1-mer incorporations in the control fragment key sequences.

  • Mean AQ20 Read Length (bp): average length, in base pairs, at which the error rate is ≤1% for all aligned reads of CF-1.

  • Percent Reads: number of all usable library reads that aligned with the CF-1 sequence divided by the total number of addressable wells.
