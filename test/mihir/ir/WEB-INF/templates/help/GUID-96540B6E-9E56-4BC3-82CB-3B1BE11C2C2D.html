Managing references (Manager/Administrator)

In the tab > References screen, Manager- and Administrator-level users can view, manage, and/or create the following reference files:

  • Reference sequence FASTA files

  • DNA panel BED files

  • Hotspot BED files

  • DNA barcode sequences

  • Fusion reference FASTA files

  • Fusion panel BED files

  • Reporting Gene Lists

  • Control fragment sequences

  • Primer FASTA files

Under each subtab, the items on each page can be sorted by clicking on column headers and the content limited by selecting from the Show drop-down lists.

  • Manage hotspot files
