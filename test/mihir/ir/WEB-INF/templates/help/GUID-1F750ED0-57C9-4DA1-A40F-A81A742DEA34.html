Summary tab

The Summary tab in the Results report window displays the following information for the run:

Field

Description

Run and Configuration Summary

Run Name

The name of the run.

Reference Genome

The reference genome used for analysis.

Targeted Regions

The name of the targeted regions BED file used.

Hotspots

The name of the Hotspot regions BED file used.

Fusion Reference

The name of the fusion reference used for analysis when applicable.

Fusion Panel

The name of the fusion panel used when applicable.

Field

Description

Amplicon Summary

Number of Targets

The number of amplicons in the panel.

On Target Reads

The number of reads mapped to the target amplicons.

Percent On Target Reads

The percentage of reads mapped on target to total reads.

Percent Full Length On Target Reads

The percentage of full length reads mapped on target to total reads.

Average Coverage

The ratio of the depth of coverage at each base in the target region to the length of the target region.

Target Coverage at 20X

The percentage of base positions with depth of coverage ≥20X in the target region.

No Strand Bias

The percentage of bases with Strand Bias between 30–70% to the total number of bases.

Coverage Uniformity

The ratio of passed number of targets to total number of targets. Where passed number of targets is the number of amplicons that have at least 0.2X mean coverage.

Field

Description

Variant Summary

Number of Fusions

The total number of fusion calls.

Number of SNVs/MNVs

The number of heterozygous single- and multi-nucleotide variations (SNVs/MNVs).

Number of INDELs

The number of heterozygous insertions or deletions.

Number of LONGDels

The number of homozygous insertions or deletions.

Number of CNVs

The number of copy number variations.
