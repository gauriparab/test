Manage reference sequences

The tab > References screen > Reference subtab lists the DNA reference sequence files available in the software.

To add a new reference file, click Add New. Enter the information in the fields and select the FASTA file containing the reference sequence information. Click Save when done.
Note:

Extremely large files take time to upload and may not appear immediately.
