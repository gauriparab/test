About the data output directory

Note:

The Data Output Directory is locked and cannot be changed. Users should configure their LIMS to access this folder location to receive sequencing output files.

The data output directory is the primary directory where outbound data from the Ion PGM™ Dx System is stored. After the sequencing run and data analysis has completed, and the report is signed off, then the software creates a separate results folder for each sample under the data output directory using the following naming convention:

AssayDev_ <Library Name>_<Assay Name>_<PlannedRunShortCode>_<PlannedRunShortCode/ReAnalysisAssay ID>
