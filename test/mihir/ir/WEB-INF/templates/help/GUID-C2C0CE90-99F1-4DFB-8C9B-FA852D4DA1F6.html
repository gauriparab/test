Switch software mode (administrator only)

The Torrent Suite™ Assay Development Software will not allow mode switching while an instrument run or data analysis is in progress.

  1. Click Sign Out to exit the software mode you are currently using.
  2. Select the software mode radio button you want to switch to under Mode Switch.
  3. Enter your software mode-specific Username and Password, then click Sign In.
    Note:

    User accounts are Mode specific. Users must set up an account in each software mode to be able to log in to the respective software mode.

    The Mode Switch dialog will appear.

  4. Read the Mode Switch guidance, then click OK.

    During mode switch "Torrent Suite is currently under maintenance! We will be back shortly." is displayed. If the screen does not automatically refresh within 10 minutes, refresh your browser.

  5. Enter your software mode-specific Username and Password, then click Sign In.
